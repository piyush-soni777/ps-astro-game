# 🚀 Astro Attack Game.

Astro Attack is an exciting space shooting game built using HTML, CSS, and JavaScript.  
In this game, you control an aircraft in space and fight against incoming enemies by shooting them and surviving as long as possible.

## ✨ Features
- 🚀 Space-themed shooting gameplay  
- ✈️ Player-controlled aircraft movement  
- 🔫 Shooting mechanics to destroy enemies  
- 👾 Multiple enemies and attack patterns  
- 💥 Smooth animations and effects  
- 📱 Responsive design (works on different screen sizes)  

---

## 🕹️ How to Play
- ⬅️ ➡️ Move the aircraft left and right  
- 🔫 Shoot enemies to score points  
- 💣 Avoid enemy attacks  
- 🏆 Survive and achieve the highest score
